📤 Βήμα 1: Upload Files

Πιο εύκολος τρόπος - Netlify Drop:

  1. Επιλέξτε όλα τα αρχεία του project
  2. Πηγαίνετε στο Netlify Drop
  3. Drag & Drop όλα τα αρχεία
  4. Θα πάρετε URL: https://random-name.netlify.app
⚠️ Σημαντικό: Βεβαιωθείτε ότι το αρχείο index-simple.html είναι στο root folder!

📦 Βήμα 2: Generate APK

Με PWA2APK (Προτεινόμενο):

  1. Πηγαίνετε στο PWA2APK
  2. Paste το Netlify URL που πήρατε
  3. App Name: Promodoro Timer
  4. Package Name: com.yourname.promodoro
  5. Κάντε κλικ "Generate APK"
  6. Download το APK! 🎉

📱 Βήμα 3: Install στο Android

  1. Ενεργοποιήστε Unknown Sources:
    Settings → Security → Install unknown apps → Chrome → Allow
  2. Download το APK στο κινητό
  3. Tap στο αρχείο → Install
  4. Έτοιμο! 🎉

🔧 Alternative - PWABuilder

Αν το PWA2APK δεν δουλεύει:

  1. Πηγαίνετε στο PWABuilder
  2. Paste το URL σας
  3. Κάντε κλικ "Build My PWA"
  4. Επιλέξτε Android → Generate Package
  5. Download το APK bundle

🎯 Quick Tips:

  • PWA Installation: Μπορείτε επίσης να προσθέσετε την εφαρμογή στην αρχική οθόνη απευθείας από το Chrome (Add to Home Screen)
  • Auto-updates: Το PWA ενημερώνεται αυτόματα, το APK χρειάζεται manual update
  • File size: PWA είναι πιο γρήγορο, APK είναι πιο "native" feeling
