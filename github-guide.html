📝 Βήμα 1: Δημιουργία Repository

  1. Πηγαίνετε στο GitHub.com/new
  2. Repository name: promodoro-timer
  3. Visibility: Public ✅ (απαραίτητο για GitHub Pages)
  4. Initialize: Αφήστε όλα unchecked
  5. Κάντε κλικ "Create repository"
📷 Screenshot: Η σελίδα δημιουργίας repository θα δείχνει τα fields που χρειάζεται να συμπληρώσετε

📤 Βήμα 2: Upload Αρχεία

⚠️ Σημαντικό: Βεβαιωθείτε ότι έχετε όλα τα αρχεία στον φάκελο σας!

🖱️ Μέθοδος A: Drag & Drop (Εύκολο)

  1. Στο νέο repository, κάντε κλικ "uploading an existing file"
  2. Επιλέξτε όλα τα αρχεία από τον φάκελο "promodoro app"
  3. Drag & Drop στο browser
  4. Commit message: "🍅 Initial Promodoro Timer upload"
  5. Κάντε κλικ "Commit changes"

💻 Μέθοδος B: Git Command Line

Κάντε διπλό κλικ στο github-deploy.bat αρχείο!

📷 Screenshot: Η σελίδα upload θα δείχνει drag & drop area για τα αρχεία

🌐 Βήμα 3: Ενεργοποίηση GitHub Pages

  1. Στο repository σας → Settings tab (πάνω δεξιά)
  2. Scroll down στο αριστερό menu → "Pages"
  3. Source: Deploy from a branch
  4. Branch: main ✅
  5. Folder: / (root) ✅
  6. Κάντε κλικ "Save"

🎉 Success! Θα δείτε μήνυμα:

✅ Your site is published at:
https://yourusername.github.io/promodoro-timer
⏱️ Περίμενε: Χρειάζονται 2-5 λεπτά για να ενεργοποιηθεί το site!

📱 Βήμα 4: Δημιουργία APK

🔗 Πάρτε το URL σας:

https://yourusername.github.io/promodoro-timer

📦 Generate APK:

  1. Πηγαίνετε στο PWA2APK.com
  2. PWA URL: Paste το GitHub Pages URL σας
  3. App Name: Promodoro Timer
  4. Package Name: com.yourusername.promodoro
  5. App Version: 1.0.0
  6. Κάντε κλικ "Generate APK"
  7. Download το APK! 🎉

📱 Βήμα 5: Εγκατάσταση στο Android

  1. Enable Unknown Sources:
    Settings → Security → Install unknown apps → Chrome → Allow
  2. Download APK στο κινητό (email, cloud, USB)
  3. Tap στο APK αρχείο → Install
  4. Έτοιμο! 🎉 Η εφαρμογή θα εμφανιστεί στην αρχική οθόνη

🎯 Advanced Tips:

  • Updates: Αλλάξτε αρχεία → Commit → APK auto-updates!
  • Custom Domain: Μπορείτε να προσθέσετε δικό σας domain
  • Analytics: Προσθέστε Google Analytics
  • PWA Direct: Χρήστες μπορούν να κάνουν "Add to Home Screen" απευθείας από το browser

🆘 Troubleshooting:

  • 404 Error: Ελέγξτε ότι το index-simple.html είναι στο root
  • PWA2APK Error: Δοκιμάστε PWABuilder
  • APK δεν εγκαθίσταται: Ενεργοποιήστε "Install unknown apps"
  • Site δεν φορτώνει: Περιμένετε 5-10 λεπτά για GitHub Pages activation
