🔧 Testing Tools

🌐 Browser DevTools:

  1. Ανοίξτε την εφαρμογή στο Chrome/Firefox
  2. Πατήστε F12 για DevTools
  3. Κάντε κλικ στο Device Toolbar icon (📱)
  4. Δοκιμάστε διάφορες αναλύσεις:
• iPhone SE (375x667)
• iPhone 12 Pro (390x844)
• Samsung Galaxy S21 (360x800)
• iPad (768x1024)
• Custom: 320x568 (μικρότερη οθόνη)

📱 Real Device Testing:

  • Ανοίξτε σε πραγματικό κινητό/tablet
  • Δοκιμάστε σε WiFi και mobile data
  • Ελέγξτε σε διαφορετικά browsers
  • Δοκιμάστε landscape/portrait orientation

✅ Mobile Functionality Checklist

🎨 Visual & Layout:

  • Το timer είναι ευκρινές και μεγάλο
  • Τα κουμπιά έχουν κατάλληλο μέγεθος (44px+)
  • Το text είναι ευανάγνωστο
  • Δεν υπάρχει horizontal scrolling
  • Τα inputs είναι εύκολα προσβάσιμα
  • Η εφαρμογή χωράει σε μικρές οθόνες

👆 Touch Interactions:

  • Όλα τα κουμπιά ανταποκρίνονται στο tap
  • Δεν υπάρχει accidental zoom στα inputs
  • Τα κουμπιά έχουν visual feedback
  • Δεν υπάρχει delay στο tap
  • Haptic feedback λειτουργεί (αν υποστηρίζεται)

🔄 Responsive Behavior:

  • Λειτουργεί σε portrait mode
  • Λειτουργεί σε landscape mode
  • Ο timer ανα-προσαρμόζεται στο screen size
  • Τα κουμπιά παραμένουν accessible
  • Το content δεν "σπάει" στις γωνίες

⚡ Performance Testing

🎯 Lighthouse Test:

  1. Ανοίξτε Chrome DevTools
  2. Πηγαίνετε στο "Lighthouse" tab
  3. Επιλέξτε "Mobile" device
  4. Κάντε κλικ "Generate report"
⚠️ Target Scores:
  • Performance: 90+
  • Accessibility: 95+
  • Best Practices: 90+
  • PWA: 90+

📊 Loading Speed:

  • Εφαρμογή φορτώνει σε <3 δευτερόλεπτα
  • Timer είναι αμέσως ορατό
  • Δεν υπάρχει layout shift
  • Icons/fonts φορτώνουν γρήγορα

📱 PWA Features Testing

🏠 Add to Home Screen:

  1. Ανοίξτε την εφαρμογή σε mobile browser
  2. Αναζητήστε "Add to Home Screen" option
  3. Επιλέξτε και προσθέστε
  4. Ελέγξτε ότι το icon εμφανίζεται
  5. Ανοίξτε από το home screen
  6. Επιβεβαιώστε ότι τρέχει σε standalone mode

🔔 Notifications:

  • Browser ζητάει permission για notifications
  • Timer completion παίζει ήχο
  • Notification εμφανίζεται (αν υποστηρίζεται)
  • Vibration λειτουργεί (Android)

💾 Offline Functionality:

  • Service Worker εγκαθίσταται
  • Εφαρμογή λειτουργεί χωρίς internet
  • Data persist όταν κλείνετε τη σελίδα
  • Settings αποθηκεύονται local

🐛 Common Issues & Solutions

❌ Problem: Buttons too small on mobile

✅ Solution: min-height: 44px (iOS), min-width: 44px

❌ Problem: Text too small to read

✅ Solution: font-size >= 16px, clamp() for responsive text

❌ Problem: Zoom on input focus (iOS)

✅ Solution: font-size: 16px στα inputs, viewport meta tag

❌ Problem: App doesn't work offline

✅ Solution: Service Worker registration, cache strategy

❌ Problem: Timer not visible in landscape

✅ Solution: @media (orientation: landscape) queries

❌ Problem: Poor performance on low-end devices

✅ Solution: CSS optimization, reduce animations, lazy loading

🧪 Quick Test Sequence

  1. Visual Test: Ανοίξτε σε κινητό → ελέγξτε layout
  2. Touch Test: Tap όλα τα κουμπιά → επιβεβαιώστε response
  3. Timer Test: Ξεκινήστε timer → ελέγξτε countdown
  4. Rotation Test: Περιστρέψτε συσκευή → ελέγξτε responsive
  5. Input Test: Προσθέστε task → ελέγξτε functionality
  6. PWA Test: Add to home screen → ανοίξτε από εκεί
  7. Notification Test: Αφήστε timer να τελειώσει
  8. Offline Test: Απενεργοποιήστε internet → δοκιμάστε app
